Enjoys Voice — Widget local test

Loads widget.js and boots the click-to-call widget against your local API.

You opened this with file://, so the browser sends Origin: null and the API will reject the key. Serve this folder over HTTP instead — e.g. python -m http.server 5500 — and open http://localhost:5500/index.html.
Before you start
  1. Node API running on :3001, web app on :4500, and the dev coturn up (for media).
  2. In Admin → API Keys, create a key with a destination (your phone number or extension).
  3. Add this page's origin — to that key's Allowed Origins.
  4. Paste the publishable key (pk_…) below and click Load widget.
