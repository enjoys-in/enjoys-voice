Quick Start:

1. Start API: cd api && bun run dev
2. Start Web: cd api/web && npm run dev
3. Open http://localhost:3000
4. Login with user1 / pass123
5. Open 2nd tab, login as user2 / pass123
6. Call each other using extension numbers
